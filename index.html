Bumpslide JS

HTML5-inspired JavaScript modules by David Knape

  • animation.js - wrapper around paulirish-style requestAnimFrame
  • bindable.js - bindable model with property change callbacks
  • canvasUtil.js - canvas drawing and image cropping utilities
  • dispatcher.js - jquery-powered event behavior decorator / event bus
  • mediaLoader.js - asset loader, queue, and cache built for HTML5 video, audio, and images
  • mouseWatcher.js - watches the mouse, so you don't have to
  • stats.js - wrapper around mr. doob's stats display
  • timeline.js - media timeline with support for frame-based ticks and seeking
  • view.js - barebones view component base

Source code structured using the AMD format with RequireJS
External dependencies: Underscore and jQuery
Unit Tests powered by QUnit
