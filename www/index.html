Bumpslide JS

This is the work of David Knape.

  • animation.js - wrapper around requestAnimFrame
  • bindable.js - a basic bindable model
  • canvasUtil.js - some canvas drawing utilities
  • dispatcher.js - jquery-powered event bus, can also add event methods to any object
  • mediaLoader.js - asset loader designed for HTML5 video, audio, and images
  • mouseWatcher.js - watches the mouse
  • stats.js - wrapper around mr. doob's stats display
  • timeline.js - media timeline with support for frame-based ticks and seeking
  • view.js - barebones view component base
